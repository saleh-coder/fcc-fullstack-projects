# Hotel Feedback Form

## Overview
The goal of this exercise was to practice building an HTML form using semantic elements and proper input types. The form simulates a feedback page for a hotel where users can submit their experience and evaluation.

## What I Practiced
- Basic HTML structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- Semantic elements: `<header>`, `<main>`, `<form>`, `<fieldset>`, `<legend>`, `<label>`, `<textarea>`
- Input types:  
  - Text (`<input type="text">`)  
  - Email (`<input type="email">`)  
  - Number (`<input type="number">`) with min/max validation  
  - Radio buttons (`<input type="radio">`)  
  - Checkboxes (`<input type="checkbox">`)  
  - Dropdown menus (`<select>`, `<option>`)  
- Form attributes: `action`, `method`, `required`, `placeholder`
- Proper form grouping with `<fieldset>` and `<legend>` for accessibility
- Submit button functionality (`<button type="submit">`)

## File Structure
- hotel-feedback-form/  
  - index.html  
  - README.md  

## Preview
To view the project locally, open `index.html` in a web browser.  
Submitting the form will send data to the FreeCodeCamp feedback submission URL (`https://hotel-feedback.freecodecamp.org`).
